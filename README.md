# Smart Parking Reservation System

A database-driven parking reservation system built for our Database Systems course at Washington State University.

## Team
- Shaya Arya
- Eduardo Melgoza
- Jamil Staten

## About
Users can register an account, add their vehicles and book parking slots in advance. Admins manage slot availability, oversee bookings, and handle disputes. Payments are tied automatically to each booking.
